# BunkerBoss Web

A simple and an intiuitive application to track my attendance in college!!

## Tech Stack

### Front-end

- React
- Tailwind CSS
- Typescript
- Vite

### Back-end

- Firebase

## Features

- **Dashboard**: Track overall attendance with a glance using stats and a dynamic radar chart.
- **Subject Management**: Add, edit, delete, or freeze subjects if no more classes are held.
- **Inline Marking**: Quickly mark present, absent, or cancel directly from the subject card.
- **Undo System**: Safely reverse the last attendance mark if you made a mistake.
- **Secure**: Authentication and backend protected by Firebase Security Rules and App Check.

## Local Setup

1. Clone the repository

    ```bash
    https://github.com/K-Surya/BunkerBoss-Web.git
    cd BunkerBoss-Web
    ```

1. Install dependencies:

   ```bash
   npm install
   ```

1. Start the development server:

   ```bash
   npm run dev
   ```
